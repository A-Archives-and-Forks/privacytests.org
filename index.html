Browser Privacy Shootout

Arthur Edelstein, 2018

This project allows browsers to be tested for fingerprintability via information leaks in certain JS and CSS APIs. We are simultaneously developing a series of mitigations that plug some of these leaks. The fingerprinting tests also serve as unit tests for the mitigations. The project is currently under development and not near completion.

Current unit tests can be run by visiting these pages:

  • test_protected.html
  • test_unprotected.html
